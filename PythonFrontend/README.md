
# Inventory Order Management Frontend

A modern React-based Inventory Order Management System that helps businesses manage products, customers, orders, and inventory efficiently.

## 🚀 Features

- Product Management
  - Add, update, delete products
  - Track inventory stock levels
  - SKU management

- Customer Management
  - Add and manage customers
  - View customer information

- Order Management
  - Create new orders
  - Track order status
  - View order history

- Dashboard
  - Inventory overview
  - Product statistics
  - Order tracking

- Responsive Design
  - Mobile-friendly UI
  - Modern user experience

## 🛠️ Technologies Used

- React.js
- TypeScript
- Vite
- Axios
- React Router
- Tailwind CSS
- Context API

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/manishpeshane011/inventory-order-management.git
```

### Navigate to project folder

```bash
cd inventory-order-management
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

Application will run at:

```bash
http://localhost:5173
```

## 🔗 Backend API

This frontend connects to the Inventory Management Backend API.

Example API URL:

```env
VITE_API_URL=https://inventory-backend1-vpd8.onrender.com/api
```

Create a `.env` file in the project root:

```env
VITE_API_URL=https://inventory-backend1-vpd8.onrender.com/api
```

## 📦 Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
